Matemática:

2.1 conjuntos Numéricos:

O Estudo da matemática se inicia com a compreensão dos conjuntos numéricos. Cada conjunto representa uma evolução na capacidade humana de quantificar, medir e calcular, englobando o anterior e resolvendo probleama que o conjunto anterior não conseguia.

2.1.1 -  Números Naturais (N)

É o conjunto mais básico, utilizado para a contagem dos elementos. Inclui o zero e todo os números inteiros positivos.

    ° Representação: { 0,1,2,3,...}
    ° Subconjunto (Naturais não-nulos): N*= { 1,2,3,4,5....} (exclui o zero).

2.1.2 Números Inteiros (Z)

Surge da necessidade de reprensentar dívidas ou temperaturas negativas. inclui todos os números naturais e seus opostos (negativos).

    ° Representação: Z = {...-3,-2,-1,0,1,2,3,... }
    ° Subconjuntos: Este conjunto possui diversos subconjuntos importantes:
Forma Trigonométrica ou Polar (z= ρ(cosθ + i)senθ)

    Símbolo     Sgnificado              Representação
      Z*        Inteiros não nulos      Exclui o 0.
      Z+        Inteiros não negativos  {0,1,2,3,...} (Idêntio ao N)
      Z-        Inteiros não positivos  {...,-2,-1,0}
      Z*+       Inteiros positivos      {1,2,3,...} (Idêntico ao N*)
      Z*-       Inteiros negativos      {...,-3,-2,-1}

2.1.3 Números Racionais (Q)

Surge da necessidade de representar partes de um todo (divisões, frações). Um número é racional se pode ser escrito na forma de uma fração a/b, onde 'a' e 'b' são números inteiros, com 'b'diferente de zero.

Os números racionais incluem:

    1.Todos os Números Inteiro: pois podem se escritos como fação (Ex.: 5 = 5/1).
    2.Decimais exatos (Finitos): (Ex.: 0,25 = 1/4).
    3.Dízimas Periódicas (Simples ou Compostas): Decimaos infinitos que repetem um padrão.
        ° Simples: O peíodo (repetição) começa logo aós a vírgula (Ex.:  0,333...=1/3).
        ° Composta: Existe um "anti-período" antes da repetição (Ex.: 0,1666 = 1/6)

2.1.4 Números Reais (R)

Para entender os Reias, precisamos definir o seu oposto:

    ° Números Irracionais (I): São números que não podem ser escritos como uma fração a/b. São dízimas inifnitas e não periódicas (não possuem um padrão de reptição).
        °Exemplos: O Número Pi (TT = 3,141592....), o número Euler (e =2,718281....), e a maioria das raízes não exatas (√2 = 1,414213...,√3,etc.).

O conjunto dos Números Reais (R) é a união do conjunto dos números Racionais (Q) com o conjunto dos números irracionais (I).

A relação de inclusão entre os conjuntos é:

N ⊂ Z ⊂ Q ⊂ R

(Os Naturais estão contidos nos Inteiros, que estão contidos nos Racionais, que estão contidos nos Reais)

2.1.5 Números Complexos (C)

Surge da necessidade de resolver raízes quadradas de números negativos (ex.:√-1), que não existem no conjunto dos números reais.

O conjunto dos Números Complexos (C) é o mais completo de todos, contendo tos os números reais. A base desse conjunto é a unidade imaginária i, definida como:

i=√-1(ou i²=-1)

Forma Algébrica (z=a+bi)

Todo número complexo z pode ser escrito nesta forma, onde :
    °a: É a parte real do número (Re(z)).
    °b: É a parte imaginária do número (Im(z)).
    °i: É a unidade imaginária.

    Ex.: No número z=3+4i, a parte real é 3 e a parte imaginária é 4.

Forma Trigonométrica ou Polar (z= ρ(cosθ + i)senθ)

É uma forma alternativa de reoresentar o número complexi , usando sua representação no plano (Plano de Argand-Gauss).

    ° ρ (Módulo): É distância do número Z até a origem (0,0). É calculado por Pitágoras ρ = √(a² + b²).
    ° θ (Argumento): É o ângulo que a linha (vetor) que liga a origem ao ponto z faz com o eixo real (horinzontal), medido no sentido anti-horário.

2.1.6 - Operações, Propriedades e Aplicações

Operações em R

O conjunto dos números reais é fachado para as quatro operações básicas (exceto divisão por zero). As principais propriedades são:

    ° Comutativa: A ordem não importa (a+b = b+a; a.b = b.a).

    ° Associativa: O agrupamento não importa ((a+b) + c = a + (b+c)).

    ° Elemento Neutro: O 0 para adição (a+0 = a) e o 1 para multiplicação (a.1 = a).

    ° Elemento Inverso (Oposto): Todo número 'a' tem um oposto '-a'(a+(-a) = 0) e todo 'a'(não-nulo) tem um inverso '1/a'(a.1/a = 1).

    ° Distributiva (Chuveirinho):a.(b+c) = ab + ac.

Aplicações Comuns (Foco de Prova):

    ° Operações com Frações (Racionais):
        A soma e subração exigem o Mínimo Múltiplo Comum (MMC) dos denominadores.
    ° Potenciação e Radiciação (Reais):
        Propriedades de potências (multiplicação de mesma base, potência de potência) e regras de simplificação de radicais são fundamentais.
